电脑配置

EFI 信息

  • 配置: i7-9750H/UHD630/AX200
  • 屏蔽 PM981 补丁
  • 电源管理/蓝牙/WIFI 状态良好，睡眠正常
  • 触摸板支持多指操作
  • 屏蔽独显，不支持 HDMI(以咨询过联想客服 HDMI、DP、Type-C 都是连接的独显)
  • 支持 FN 热键，可以使用系统组合键调节亮度、声音等
  • 有任何问题下方加群解决
